# Sample Import Files

本目录提供阶段 3 的示例导入文件。

- `annual_exam_sample.csv`
- `blood_test_sample.csv`
- `body_scale_sample.csv`
- `activity_sample.csv`
- `activity_invalid_sample.csv`

模板文件位于 `templates/`：

- `templates/annual_exam_template.csv`
- `templates/blood_test_template.csv`
- `templates/body_scale_template.csv`
- `templates/activity_template.csv`

当前导入器支持：

- `.csv`
- `.xlsx`
- `.xls`

如果使用 Excel，请保持同样的表头，导入器会默认读取第一个工作表。
